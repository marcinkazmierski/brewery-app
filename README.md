# brewery-app - Zdalny browar
Flutter mobile application.

## installation/run
- copy .env.dist to .env
- generate `android/app/google-services.json` from firebase

## web version:
- flutter doctor
- flutter devices
- export CHROME_EXECUTABLE=/snap/bin/chromium
- flutter run -d chrome
